# Heart Disease Prediction using Machine Learning

This project predicts the risk of heart disease using a Logistic Regression model and displays the result through a Flask web application.

## Features
- Predicts heart disease risk based on:
  - Age
  - Cholesterol
  - Blood Pressure
  - Heart Rate
- Displays risk percentage
- Frontend dashboard with HTML and CSS
- Flask backend integration

## Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn
- HTML
- CSS

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
